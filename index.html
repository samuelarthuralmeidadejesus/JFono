// contexto

O problema que o JFono resolve

O acompanhamento vocal é frequentemente feito de forma manual, dificultando a organização e comunicação entre profissionais e pacientes. O JFono propõe uma solução digital para fonoaudiólogos e clínicas.

🎙️

Monitoramento Vocal

Centraliza exercícios vocais com descrições, sons e tempo mínimo de execução para cada paciente.

🏥

Gestão de Clínicas

Organiza dados institucionais de clínicas de fonoaudiologia com CNPJ e identificação única.

👤

Perfis Completos

Modela pacientes e profissionais com atributos, contatos e métodos de atualização de dados.

// código

Exemplo de uso

Instanciação e manipulação das classes principais do sistema, demonstrando construtores sobrecarregados e encapsulamento.

App.java 
// Exercícios vocais com construtores sobrecarregados
Exercicio ex1 = new Exercicio("Soprar Canudo", "Relaxar cordas vocais", "/", 3, "/");
Exercicio ex2 = new Exercicio("Vibração Labial", "Vibrar lábios", "brrrrr");
Exercicio ex3 = new Exercicio("Canto Sustentado");

// Clínicas
Clinica c1 = new Clinica("Voz & Saúde", "12.345.678/0001-99");

// Pacientes com setter
Paciente pac = new Paciente();
pac.setNome("Lucas Oliveira");
pac.setEmail("lucas@email.com");

// Profissional
Profissional p1 = new Profissional("Dr. João Lima", "(11) 98765-4321");
